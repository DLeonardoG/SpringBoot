1. Qu son los frameworks?

ALT
Los frameworks son marcos de trabajo o conocidos por ser entornos de trabajo que buscan apoyar el desarrollo de aplicaciones profesionales que sean estables y dinmicas. Estos Frameworks vienen con un conjunto de paquetes (libreras), herramientas y diversas utilidades que buscan apoyar el desarrollo de un macro-proyecto.
Un framework busca ser parte del paradigma del desarrollo de software gil , con el fin de desarrollar proyectos de manera productiva y en menos tiempo. En este caso, desarrollar aplicativos web que tengan alta complejidad de consumo a nivel de informacin y contenido ser de gran ayuda al momento de implementar frameworks en el proyecto.
Las caractersticas principales de un framework son las siguientes :
Escalabilidad : Un proyecto basado en un framework permite crearlo al ritmo y necesidades del cliente en el momento que sea necesario.

Modelo Vista - Controlador (MVC): Al estar ligado al modelo vista-controlador permitir tener una estructura estandarizada, no solamente para manejar los elementos internos sino tambin para consumir correctamente la informacin proveda
Minimizar la escritura de cdigo repetitivo: Gracias a los elementos anteriormente mencionados, se puede minimizar la creacin de cdigo anteriormente establecido, pues su utilidad es referenciar dicho cdigo para su posterior implementacin.
Bases generales auto-gestionadas: Elementos tales como la seguridad, manejo de informacin, implementacin de vistas sern elementos que podrn ser gestionados con mayor facilidad desde un framework.

4. Qu es Spring Boot?

ALT
Spring Boot es un marco de desarrollo avanzado que ha revolucionado el panorama de desarrollo de aplicaciones Java empresariales. Diseado para simplificar el proceso de creacin, configuracin y despliegue de aplicaciones, Spring Boot se destaca por su enfoque en la convencin sobre la configuracin, lo que significa que los desarrolladores pueden concentrarse en la lgica de negocio sin la carga de configuraciones extensas. Al utilizar anotaciones y proporcionar una estructura de proyecto bien definida, Spring Boot elimina gran parte de la complejidad asociada con la configuracin manual de un proyecto Spring tradicional.
Una caracterstica distintiva de Spring Boot es su capacidad para empaquetar aplicaciones como archivos ejecutables independientes, ya sea en formato JAR o WAR, lo que simplifica significativamente el despliegue y la gestin de dependencias. Adems, Spring Boot ofrece una amplia integracin con tecnologas modernas, como Spring Data JPA para el acceso a bases de datos, Spring Security para la implementacin de medidas de seguridad, y Thymeleaf para el desarrollo de vistas en aplicaciones web.
Al igual que esto hay otras caractersticas representativas las cuales son las siguientes:
Configuracin Automtica: Spring Boot realiza la configuracin automtica, lo que significa que intenta configurar la aplicacin basndose en las dependencias presentes en el proyecto. Esto reduce la cantidad de configuracin manual que un desarrollador necesita realizar.
Incrustacin de Servidor: Spring Boot incluye servidores integrados como Tomcat, Jetty o Undertow, lo que significa que no es necesario configurar un servidor por separado para ejecutar la aplicacin. Puedes empaquetar la aplicacin como un archivo JAR ejecutable o un archivo WAR para implementarla en un servidor.
Inicio Rpido: Spring Boot facilita la creacin de aplicaciones con un inicio rpido. Con unas pocas anotaciones y configuraciones mnimas, puedes desarrollar aplicaciones funcionales de manera rpida.
Microservicios: Spring Boot es ampliamente utilizado en el desarrollo de arquitecturas de microservicios. Facilita la creacin de servicios independientes que se pueden implementar y escalar de manera independiente.
Gestin de Dependencias: Spring Boot utiliza Spring Boot Starter, que son dependencias preconfiguradas para varias tecnologas y frameworks. Esto facilita la inclusin de las dependencias necesarias en tu proyecto.
Monitorizacin y Actuadores: Spring Boot incluye un conjunto de actuadores que proporcionan caractersticas de monitorizacin y administracin, como la informacin sobre la aplicacin, la salud del sistema y la gestin de los beans de Spring.
Soporte para Spring Ecosystem: Spring Boot se integra bien con otros proyectos del ecosistema Spring, como Spring Data, Spring Security y Spring Cloud, para facilitar el desarrollo de aplicaciones ms completas.
